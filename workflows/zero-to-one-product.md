# Workflow: Zero To One Product Delivery

Use when starting or reshaping a product.

## Sequence

1. Product Agent: define user job, target user, scope, non-goals, success metrics.
2. UEAgent: define product shape, primary workflow, screen map, states.
3. Architecture Agent: define smallest sufficient architecture, boundaries, contracts.
4. Engineering Agent: implement first thin slice.
5. Code Review Agent: review for regression, complexity, security, tests.
6. Test Agent: validate core path and recovery path.
7. DevOps Agent: define deploy and rollback.
8. Operator Agent: create demo, docs, and next iteration summary.

## Output Pack

- PRD
- UE spec
- Architecture note
- Implementation diff
- Review report
- Test report
- Release runbook
- Demo script
- Next iteration plan
